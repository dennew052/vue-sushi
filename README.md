# vue-sushi

Этот проект создан на Vue 3

Шаблон интернет-магазина, который позволяет добавлять товары в корзину, оформлять покупки, искать и фильтровать товары

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
